Advanced Capabilities
Beyond the basics —
powerful tools built in
Capabilities that go well beyond a standard helpdesk tool, built natively without additional cost or infrastructure.
Last Logon Location Finder
Query the PDC Emulator for a user's most recent remote logon event (ID 4624) to identify the IP address and hostname of the last machine they authenticated from — essential for tracing stale credential sources causing repeat lockouts.
Remote BIOS Password Provisioning
Set the BIOS/UEFI administrator password on Lenovo, Dell, and HP machines using manufacturer WMI providers — remotely, without opening a session. Checks current status first and only acts when no password is set, preventing lockouts.
Remote Quick Actions
One-click remote operations from the context menu: Force Group Policy Update, Restart Print Spooler, Flush DNS, Send popup message to the active user, and remote Reboot or Shutdown — all without a full remote desktop session.
Modern App (AppX) Software Management
The software inventory enumerates both traditional Win32 apps and modern AppX/UWP packages on remote machines, with silent uninstall support for both — without triggering WinRM WinRT crashes common in other remote management tools.
Real-Time Printer Health Checks
The Printer Manager runs parallel ping and port-9100 TCP health checks against every printer's IP simultaneously, stamping each row with Online, Ping OK, or Offline status so you know printer reachability before starting any work.
In-App Management Dashboard
An analytics dashboard built directly into the application shows current-month KPIs, a 14-day unlock trend chart, and a top-5 most-locked users chart — all computed live from the same audit logs, no external reporting tool needed.

What's Next
Planned improvements
The IT engineering team has identified the following enhancements for upcoming development cycles.
Management Reporting Export
Add weekly and monthly summary report exports showing unlock counts, incident metrics, and technician activity for IT leadership. The in-app dashboard already computes these metrics; a PDF/CSV export button is the primary remaining work.
Ticket Feed Enhancements
Expand the Recent Tickets panel with ticket detail view, ability to add a public note directly from the feed, and inline status change without opening the Freshservice web interface.
Remote Screenshot
Capture the active desktop of a remote machine without a full remote session. Currently blocked by Windows Session 0 isolation — WinRM runs in a non-interactive context with no display handle. Under evaluation for alternative approaches.

Add a quick-launch button to initiate Splashtop SOS remote support sessions directly from the computer context menu. The UI placeholder is already in place.
